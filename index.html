SociaLite

 🛠 Pripomoček

Program za zmernejšo uporabo družabnih omrežij z omejitvijo prikazanih objav.
Trenutno je podprta le uporaba YouTube-a.

Namen

Če se tudi vam kdaj zgodi, da čas ob družabnih omrežjih kar izpuhti, čeprav bi bolj potrebovali spanca ali pa prave družbe, je ta aplikacija morda del rešitve, da ne bo več prihajalo do tovrstnih situacij:

Uporaba

Na Google Cloud Console si pridobi API ključ in ga vnesi v nastavitvah aplikacije. Nato si omogoči YouTube Data API v3. Z uporabo iskalnika ID-jev pridobi ID-je kanalov, ki jih želiš spremljati in jih dodaj v nastavitvah.

API ključ in ID-ji kanalov se shranijo lokalno v brskalniku.

Kanal lahko naknadno tudi skriješ iz prikaza ali pa ga odstraniš.

Med objavami boš tako videl zadnje tri objave izbranih kanalov razvrščene po datumu objave.

Priporočila

API ključa ne delimo z drugimi osebami (oziroma to počnemo v omejenem obsegu), saj je Google API le do neke točke brezplačen.

Za boljšo izkušnjo digitalnega minimalizma se izogibajte sledenju velikemu številu kanalov in dejansko aplikacijo na namizju/v zaznamkih raje nadomestite s to aplikacijo, da boste s tem zares zmanjšali svojo uporabo družabnih omrežij.

V branje vam priporočam tudi branje knjige Digitalni minimalizem, ki je izšla pri zavodu Iskreni.
